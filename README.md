## TwentyOne / Blackjack Console Game

The TwentyOne project is a C# console application based on the card game Blackjack. The program uses object-oriented programming concepts to create a playable card game between the player and dealer.

### Skills Demonstrated

- C# console application development
- Classes and objects
- Inheritance
- Methods and constructors
- Conditional statements
- Loops
- Lists and collections
- Debugging with breakpoints
- Step Into and Step Over debugging tools
- Reading and understanding existing code

### Project Summary

This project helped me understand how larger programs are organized into multiple classes. I used Visual Studio’s debugger to step through the program line by line and understand how the game logic works.

The program includes logic for:

- Creating a deck of cards
- Dealing cards to the player and dealer
- Calculating hand values
- Applying game rules
- Determining the winner
- Handling player choices during gameplay

This project was useful for learning how different parts of a program work together and how object-oriented programming can make a project easier to organize.

## Car Insurance Quote Application

The Car Insurance project is an MVC web application that simulates a car insurance quote system. The application collects information from the user and calculates an insurance quote based on business rules.

### Skills Demonstrated

- ASP.NET MVC
- C# business logic
- User input handling
- Model, View, Controller structure
- Form submission
- Data validation
- Quote calculation logic
- Admin page functionality

### Project Summary

This project takes user information such as age, vehicle details, driving history, and other risk factors, then calculates a quote based on the provided information.

The project also includes an admin page that allows staff to view submitted quotes.

This project helped me understand how web applications are structured and how user input can be processed through an MVC application.

## NFL Stats Application

The NFL Stats project is a data-based MVC web application designed to store and display football statistics. The application was built to practice working with databases and displaying stored information in a web application.

### Skills Demonstrated

- ASP.NET MVC
- Entity Framework 6
- Code First workflow
- Database models
- Data storage and retrieval
- Displaying records in views
- Working with sports/statistical data

### Project Summary

This project stores football-related statistics and displays them through a web application. It uses Entity Framework 6 with a Code First approach, meaning the database structure is created from C# classes.

This project helped me practice connecting a C# application to a database and organizing data so it can be viewed and managed through a website.
